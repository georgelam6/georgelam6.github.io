Quo

Single-header library, intended for use in games. Still in very early development

This library is being written from scratch in C, using only system dependencies. Currently it can render to a window on both Windows and Linux (X11) using OpenGL, and capture keyboard input.

  • quo.h - The library source itself
  • sbox.c - C file that I use for testing the library
  • quo - Main GitHub repository
